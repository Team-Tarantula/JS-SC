# Orientation

Window - DOM + BOM + JavaScript Core
- DOM (Document Object Model) - document, ...
- BOM (Browser Object Model) - navigator, screen, location, frames, history, XMLHttpRequest, ...
- JCORE - object, array, function, date, ...
